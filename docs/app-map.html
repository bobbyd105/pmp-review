Logic modules

FilePurposeConsumed by
src/quiz/quizSession.js Pure quiz-session engine (no React): random no-repeat question selection (Fisher-Yates, injectable rng), submit/advance state transitions, score and missed-question selectors, and localStorage save/load/clear with validation that rejects corrupt or stale stored sessions. Storage key: pmp-quiz-session-v1. Quiz, QuizResults; unit-tested directly

Data sources

FilePurposeSchemaConsumed by
data/questions.json Question bank seed data: 12 original PMP-style questions across all 3 ECO domains (People, Process, Business Environment). Local JSON per constitution Section 3 — no database, no API. Array of objects: id, eco_domain, eco_task, question, options[], correct_answer (exact string match to one entry in options), explanation QuestionBank, Quiz; test files
localStorage key pmp-quiz-session-v1 In-progress/completed quiz session state (per decision_log.md Decision #3): survives page reload; cleared on Start new quiz Object: questionIds[], currentIndex, results[] of {questionId, selected, correct}. Score is derived, not stored. quizSession.js only (components never touch localStorage directly)
